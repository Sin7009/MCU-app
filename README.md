# MGPU Schedule Manager (МГПУ Расписание)

Это приложение предназначено для администраторов, составляющих расписание преподавателей вуза. Оно позволяет управлять списком преподавателей, учитывать их предпочтения, загружать учебные планы и распределять нагрузку.

## Основные возможности

1.  **Управление преподавателями**:
    *   Добавление новых преподавателей (ФИО, Email).
    *   Указание предпочтений (например, нежелательные дни недели).
    *   Просмотр списка всех преподавателей.

2.  **Управление учебным планом**:
    *   Загрузка учебного плана из CSV-файла.
    *   Просмотр списка дисциплин (Часы, Группы, Семестр).
    *   Очистка текущего плана.

3.  **Распределение нагрузки**:
    *   Назначение преподавателей на конкретные дисциплины через удобный интерфейс.
    *   Отображение текущего назначения в таблице.

## Установка и запуск

### Требования
*   Python 3.x
*   Библиотека `tkinter` (обычно входит в стандартный пакет Python).

### Запуск
1.  Склонируйте репозиторий или скачайте файлы.
2.  Перейдите в корневую директорию проекта.
3.  Запустите приложение командой:
    ```bash
    python3 main.py
    ```

## Формат файла CSV для загрузки плана

Для загрузки учебного плана подготовьте CSV файл (разделитель запятая) со следующими колонками (заголовки опциональны, но порядок важен):
1.  Название дисциплины
2.  Количество часов
3.  Название группы
4.  Номер семестра

**Пример:**
```csv
Subject Name, Hours, Group Name, Semester
Высшая математика, 72, ИВТ-21, 1
Программирование, 108, ПИ-20, 2
```

Вы также можете сгенерировать пример файла прямо из приложения, нажав кнопку "Создать пример CSV".

## Структура проекта

*   `main.py` - Точка входа в приложение.
*   `src/database.py` - Логика работы с базой данных (SQLite).
*   `src/models.py` - (Зарезервировано) Модели данных.
*   `src/logic/importer.py` - Логика импорта данных (CSV).
*   `src/gui/` - Компоненты пользовательского интерфейса.
